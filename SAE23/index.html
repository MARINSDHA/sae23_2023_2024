SAé23: Mettre en place une solution informatique pour l’entreprise

Quel est l'objectif de ce site ?

Ce site a pour objectif d'exploiter les données provenant de capteurs répartis dans les bâtiments de l’IUT en proposant une interface dédiée.
De plus, pour ce faire nous avons utilisé des projets comme le "traitement de données".

Mais qu'est-ce que la SAÉ23 ?

Ce projet a pour objetif d'offrir 2 solutions : Une solution Docker et Une solution site web dynamique qui toutes les deux permettent la visualisation des données capteur.
Ces deux solutions offrent des avantages et des inconvénients:
  • Solution Docker
    Avantages : ● Prototypage rapide, ● Pas ou très peu de codage, ● Interface conviviale etclaire pour le gestionnaire
    Inconvénients: ● Pas de gestion de comptes, ● Accès Nodered non sécurisé, Nécessité de formaliser les requêtes par le gestionnaire si affichage plus précis

  • Une solution site web dynamique
    Avantages : ● Gestion plus fine des comptes et de la base, ● Utilisation de formulaire pour formaliser les requêtes (plus d’options possibles pour le gestionnaire) ● Sécurisation des accès
    Inconvénients: ● Développement plus long, ● Davantage de codes à produire

Que va présenter ce site web ?

Ce site web va permettre d'afficher les différentes parties du projet :
  • Une page administration, accessible uniquement par l'administrateur du site.
  • Une page gestion, accessible uniquement par les gestionnaires.
  • Une page consultation, accessible à tous.
  • Une page contact, pour nous contacter par mail.
  • Les mentions légales de notre site web.
